# Test Weather Backend

## Setup ENV variables

1. Create a new file named `.env`, for example like this `cp .env.example .env`.
2. Update the variables with the right values if needed.

## Commands

```bash
# Installation
yarn

# Development
yarn start:dev

# Production mode
yarn start:prod

# Check ESLint
yarn lint:check

# Check Prettier
yarn format:check

# Check all (ESLint + Prettier)
yarn check:all
```
